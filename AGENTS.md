# AGENTS

## Agent skills

### Issue tracker

Issues live in this repo's **GitHub Issues** (via the `gh` CLI). See `docs/agents/issue-tracker.md`.

### Triage labels

The `triage` skill applies the canonical defaults: `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`. See `docs/agents/triage-labels.md`.

### Domain docs

This repo uses a **single-context** layout: `CONTEXT.md` at the root and ADRs in `docs/adr/`. See `docs/agents/domain.md`.
